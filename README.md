# Proyecto Nutrición Solana
Contrato inteligente desarrollado con Anchor para el registro y evaluación antropométrica de pacientes.

## Funcionalidades
- CRUD completo (Create, Read, Update, Delete) de pacientes.
- Cálculo automático de grasa corporal (Fórmula de Yuhasz).

## Requisitos para correr
1. Tener instalado Rust y Anchor.
2. Ejecutar `anchor build` para compilar.
3. Ejecutar `anchor test` para verificar la lógica.
